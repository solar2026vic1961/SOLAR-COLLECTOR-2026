# SOLAR-COLLECTOR-2026
Двухконтурный солнечный коллектор с водяным фильтром обратного ИК-излучения. Холодная вода во внешнем слое (1-2 мм) пропускает видимый свет к чёрному экрану, но поглощает тепловое излучение от горячей воды, блокируя радиационные и кондуктивные потери. Public Domain (CC0). Дата публикации: 2025-04-08.
